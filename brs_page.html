وب‌سرویس رایگان بورس - Api رایگان بورس Tsetmc

وب‌سرویس رایگان بورس (Api رایگان بورس Tsetmc)، دیتای لحظه‌ای کلیه نمادهای بازار بورس و فرابورس (سهام، صندوق‌های سهامی، صندوق‌های درآمد ثابت، صندوق‌های طلا، صندوق‌های مختلط، صندوق‌های املاک، صندوق‌های کالایی) را از سایت tsetmc بورس تهران ارائه میکند. Api بورس دیتا را از سایت tsetmc بورس تهران جمع آوری میکند که از آن میتوانید برای مانیتورینگ بورس ایران استفاده کنید. تنها کافیست در صفحه Api Key رایگان بورس Tsetmc یک درخواست ارسال کنید.

ویژگی‌های وب‌سرویس رایگان بورس

  • دیتای لحظه‌ای تابلوی کلیه نمادهای بورسی و فرابورسی (سهام، صندوق‌های سهامی، صندوق‌های درآمد ثابت، صندوق‌های طلا، بورس کالا، بازار آتی، اوراق بدهی، اوراق تسهیلات مسکن)
  • دسترسی به دیتای دقیق و کامل قیمتی، معاملاتی، حقیقی-حقوقی، عرضه و تقاضا
  • دریافت دیتای لحظه‌ای با تاخیر پایین، از وبسایت tsetmc بورس اوراق بهادار تهران
  • نامگذاری پارامترها دقیقا مشابه با متغیرهای فیلترنویسی tsetmc
  • خروجی دیتا در قالب استاندارد JSON
  • کاملا رایگان
