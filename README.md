# TSEAnalysis - تحلیل تکنیکال بورس ایران

یک برنامه وب Flask برای تحلیل تکنیکال نمادهای بورس ایران با استفاده از API BrsApi.ir

## ویژگی‌ها

- دریافت داده‌های کندل استیک نمادهای بورس
- محاسبه ۱۵ اندیکاتور تکنیکال
- تولید نمودارهای حرفه‌ای
- رابط کاربری ساده وب

## اندیکاتورهای موجود

1. SMA 10 و 20
2. EMA 10 و 20
3. RSI 14
4. MACD (با سیگنال و هیستوگرام)
5. Bollinger Bands
6. ATR 14
7. CCI 20
8. Stochastic Oscillator
9. ROC 10

## نصب و راه‌اندازی

1. کلون کردن پروژه:
```bash
git clone <repository-url>
cd TSEAnalysis
```

2. نصب وابستگی‌ها:
```bash
pip install -r requirements.txt
```

3. اجرای برنامه:
```bash
python app.py
```

4. باز کردن مرورگر و رفتن به `http://localhost:5000`

## استفاده

1. نماد مورد نظر را انتخاب کنید
2. روی دکمه تحلیل کلیک کنید
3. فایل zip حاوی نمودارها دانلود خواهد شد

## تنظیمات

کلید API و تنظیمات دیگر در فایل `config.py` قابل تغییر است.

## APIهای پشتیبانی شده

- کندل استیک بورس
- لیست نمادها
- (قابل گسترش برای سایر APIهای BrsApi)

## نکات مهم

- **اتصال به API**: ممکن است به دلیل محدودیت‌های شبکه یا مسدودسازی IP، اتصال به BrsApi.ir برقرار نشود. در این صورت از VPN استفاده کنید.
- **کدهای نماد**: کدهای tsetmc معتبر استفاده کنید (مثل IRO1FOLD0001 برای فولاد).
- **تحلیل کندل استیک**: فعلاً فقط برای سهام در دسترس است.
