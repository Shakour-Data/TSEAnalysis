# راهنمای استفاده از وب‌سرویس BrsApi

این سند حاوی نکات حیاتی برای استفاده از APIهای پروژه Bedaan4D-ML و جلوگیری از مسدودسازی دسترسی است.

## ۱. اطلاعات دسترسی (محرمانه)
- **کلید API:** `BA9C8JBliDmfPapn9WYTX76uR5Q3m2r3`
- **وضعیت:** فعال

## ۲. قوانین استفاده منصفانه (Fair Use Policy)
- **محدودیت نرخ:** از ارسال درخواست‌های مکرر و بدون منطق خودداری کنید.
- **جریمه تخلف:** ارسال درخواست بیش از ظرفیت منجر به مسدودسازی دائمی کلید API، ایمیل و شماره موبایل می‌شود.
- **هدایت به فایل‌های حجیم:** در صورتی که تعداد درخواست‌های غیرمجاز به **۵۰۰ عدد** برسد، کلیه درخواست‌ها به فایل‌های حجیم هدایت می‌شوند که باعث کندی شدید و هزینه‌های ناخواسته می‌گردد.

## ۳. نکته حیاتی برای کاربران Python (بسیار مهم)
سیستم امنیتی **6G Firewall** وب‌سرویس، User-Agent پیش‌فرض کتابخانه‌های پایتون (مانند `requests`) را به‌صورت خودکار مسدود می‌کند.

### راه حل:
الزاماً باید از یک User-Agent معتبر مرورگر در هدر درخواست‌ها استفاده کنید. 
**نمونه کد صحیح:**
```python
headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/91.0.4472.124 Safari/537.36'
}
response = requests.get(url, headers=headers)
```
*عدم رعایت این مورد منجر به مسدودسازی IP شما خواهد شد.*
